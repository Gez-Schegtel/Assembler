# Ejercicios de Assembler - Arquitectura de Computadoras

Este repositorio contiene una colección de ejercicios de programación en lenguaje ensamblador (Assembly), desarrollados en el marco de la cátedra de Arquitectura de Computadoras de la Universidad Tecnológica Nacional (UTN) - Facultad Regional Resistencia.

## Descripción

El objetivo de este proyecto es alojar las resoluciones de las guías de trabajos prácticos y ejercicios de exámenes finales. El código está orientado a la arquitectura 8086 y documentado con fines académicos y de estudio.

## Herramientas y Requisitos

Para la escritura, compilación y emulación del código fuente se utiliza estrictamente:

*   emu8086 (Microprocessor Emulator)

> Nota: El uso de emu8086 es un requisito obligatorio establecido por la cátedra para garantizar la compatibilidad y estandarización de las instrucciones durante el cursado y las evaluaciones.

## Estructura del Repositorio

El repositorio se organiza en las siguientes carpetas según el tipo de actividad:

*   /Ejercicios de la guía: Contiene la resolución de los ejercicios propuestos en la guía de trabajos prácticos de la materia.
*   /Ejercicios de final: Incluye resoluciones de ejercicios tomados en mesas de exámenes finales anteriores, utilizados para práctica y repaso.

## Ejecución

Para ejecutar los archivos .asm contenidos en este repositorio:

1.  Abrir emu8086.
2.  Seleccionar la opción "Open" y navegar hasta el archivo deseado.
3.  Presionar el botón "Emulate".
4.  Utilizar los controles de "Run" o "Single Step" para visualizar el comportamiento de los registros y la memoria.
